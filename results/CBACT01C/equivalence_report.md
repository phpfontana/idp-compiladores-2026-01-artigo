### Optimization attempted: eliminate per-record heap allocations

**Name:** `reduce-per-record-allocations`  
**Description:** Three per-record heap allocations were eliminated:

1. `cobdatft()` previously returned a new `byte[10]` per record. Replaced with `cobdatftInPlace()` that writes directly into `outRecord`, avoiding the allocation and subsequent `arraycopy`.
2. `encodeDisplaySigned(-102500L)` was called each iteration for BAL(3), which is a constant value. Promoted to a static final constant `DISPLAY_NEG_1025_00`.
3. `writeVbrRecord()` previously allocated `byte[4]` for the RDW header on each of its 2 calls per record. Replaced with a pre-allocated static `RDW_BUF[4]`.

**Equivalence check post-optimization:** 4/4 PASS (OUTFILE, ARRYFILE, VBRCFILE, STDOUT all match golden master byte-for-byte).
